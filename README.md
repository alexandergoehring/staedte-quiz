# staedte-quiz
von Claude Opus 4.6 generiertes Browserspiel

Promt an Claude Opus 4.6:
„Erstelle ein Browserspiel.
Es soll ein Quiz werden. 
Es sollen zehn deutsche große Städte erraten werden anhand von Bildern von Sehenswürdigkeiten der Städte.“

Initialer Prompt + ca. 4 Iterationen (aufgrund von Bilder Download Problemen, Error Output zur Problemlösung an KI zur automatisierten Problemlösung einfach zurück gegeben)

Grundlogik (Städte Auswahl, Sehenswürdigkeiten Auswahl, Quiz A/B/C/D Städtevorschläge, Fortschrittsbalken, Rundenzähler, Punktezähler, finale ergebnisabhängige textuelle und grafische Auswertung) und Design war bereits beim FirstShot korrekt.

Benötigte Zeit: ca. 30min
